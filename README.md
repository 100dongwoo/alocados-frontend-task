## 알로카도스 프론트엔드 과제


## 실행 방법
```
# 깃허브에서 소스코드를 다운로드 받습니다.

git clone https://github.com/100dongwoo/alocados-frontend-task

# 프로젝트의 기존 패키지를 설치합니다.

yarn install

# 로컬 서버를 통해 프로젝트를 시작합니다.

yarn start

```
